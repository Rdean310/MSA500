# Equity Method Student Site

This is a ready-to-upload Vite + React website based on the Equity Method presentation.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Upload to GitHub and deploy on Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel and import the GitHub repository.
4. Vercel should detect Vite automatically.
5. Click Deploy.

## Files you will usually edit

- `src/App.jsx` for the website content
- `src/styles.css` for the design
